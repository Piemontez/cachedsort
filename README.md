# Cached Sort

Source code com implementação do Cached Sort em c++ e diversos testes de desempenho. 

Cached Sort source code in c ++ and several performance tests.

## Algorithms Tested

Cached Sort 1, 2 and 3

Bubble Sort

Selection Sort

Insertion Sort

Shell Sort

Heap Sort

Merge Sort

Quick Sort



## Build and Run

```bash
cmake ./
make

./cachedsort
```
